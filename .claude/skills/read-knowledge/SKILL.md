# Read Knowledge

Before starting any task, check `.opencode/knowledge/` for files relevant to the work at hand. The AGENTS.md file at the project root has a table mapping areas to knowledge files.

```bash
ls .opencode/knowledge/
```

Read any files that relate to your task. This avoids re-discovering information that has already been documented.
